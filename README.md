## 🦴 概览

- **一个拥有简易GUI 完全基于Mineflayer的免费开源我的世界Bot**

## ✨ 核心特色

- ✅ 完全免费开源且持续更新 其他Bot均价10-50不等
- ✅ 简单易懂GUI，完全没有电脑知识的小白也可以随意使用
- ✅ 兼容Windows，Macos的多个架构
- ✅ 一键安装所有依赖，甚至无需打开终端，无需输入密码
- ✅ 支持自定义配置，GUI配置仅为基础配置，开发者可以前往 /安装路径/bot.js 更改详细配置

## ⏬ 安装

- 从release页面下载后，安装java，双击打开程序即可

## 🔥 构建

./gradlew shadowJar

## 命令

**全部在仿真终端内输入，回车即可执行**

- goto x y z 比如 goto 100 64 100            
- yaw <角度> 比如 yaw 90
- action right_click / left_click

暂时只有这些，功能基本仅限挂机，做珍珠点Bot等，以后会做升级
